CSS: classes vs IDs

What are the best practices when using classes vs IDs

  • First, what are we talking about? Class and ID selectors are used to identify and style various HTML elements.

  • Classes are essentially rules we create that we can apply to style our web page. We can then apply those classes over several different HTML elements and use them repeatedly.

  • IDs are used on a specifically on a single element. IDs also have priority over classes
